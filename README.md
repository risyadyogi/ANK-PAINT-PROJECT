 ANK PAINT PROJECT Website

Professional website template for a **motorcycle paint workshop / custom paint garage**.
This project is designed for small automotive businesses that want a modern online presence.

## Features

* Modern responsive design
* Motorcycle paint service listing
* Price estimation section
* Gallery of completed paint jobs
* WhatsApp contact integration
* Booking service form
* SEO optimized structure

## Technologies

* HTML5
* CSS3
* Responsive Layout
* Mobile Friendly Design
